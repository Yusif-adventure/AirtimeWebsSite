# Frontend
## Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`

# Backend
Contains logic for `orders` and `payments`.
- Built with Deno & Supabase Edge Functions.
- Check `backend/functions/server/index.tsx` for the main logic.

# Database
- `database/schema.sql`: Contains the SQL schema for your tables.
- You can run this in your Supabase SQL Editor.
